============
Installation
============


Stable release
--------------

To install lcep-package, run this command in your terminal:

.. code-block:: console

    $ pip install lcep-package

This is the preferred method to install lcep-package, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for lcep-package can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/mlf-core/lcep-package

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/mlf-core/lcep-package/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ make install


.. _Github repo: https://github.com/mlf-core/lcep-package
.. _tarball: https://github.com/mlf-core/lcep-package/tarball/master
